# Universal-Interactive-Drawer-Cabinet-System
Universal Interactive Drawer & Cabinet System
Bu modül, oyun dünyasındaki dolap kapakları, çekmeceler ve sürgülü nesneler ile etkileşime girmeyi sağlayan evrensel bir denetleyicidir. Koda gömülü değerler barındırmaz, tamamen Inspector üzerinden özelleştirilebilir.

Özellikler:

Çift Modlu Hareket: Nesneleri ister eksen üzerinde kaydırın (Slide), ister menteşe etrafında döndürün (Rotate).

Akıllı Parlama (Highlight): Sadece belirlenen Mesh'leri parlatarak, dolap içindeki toplanabilir eşyaların yanlışlıkla parlamasını önler (HDRP uyumlu).

Olay Yönlendirmeli (Event-Driven): UnityEvent yapısı sayesinde ses çalma, görev tetikleme veya parçacık efekti oynatma gibi işlemleri ekstra kod yazmadan yapabilirsiniz.

Kurulum:

Scripti etkileşime girilecek nesneye atayın.

Movement Type seçiminizi yapın ve açılma/dönme offset (mesafe/açı) değerlerini girin.

On Drawer Opened / Closed eventlerine sisteminizdeki ses veya animasyon yöneticilerini bağlayın.
